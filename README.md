# Trump MNQ Alert Bot (Beginner Version)

A simple Python script that monitors Donald Trump's Truth Social posts and sends **Bull / Bear alerts** for MNQ (Micro Nasdaq futures) to Telegram.

### Features
- Checks every 30 seconds
- Very easy keyword-based Bull/Bear detection
- Clean Telegram alerts

### How to Use
1. Clone the repo
2. `pip install -r requirements.txt`
3. Fill in `.env` with your Telegram token and chat ID
4. Run `python main.py`

Perfect for learning Python + APIs + trading alerts!

Warning: This is for educational purposes. Not financial advice. Market reactions to Trump posts can be fast and unpredictable.
